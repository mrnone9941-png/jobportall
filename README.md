# UrNext

A full-stack job portal built with the MERN stack that helps employers post jobs and job seekers discover and apply for opportunities.

## Features
* User authentication
* Browse and search jobs
* Apply for jobs
* Post and manage job listings
* Responsive and user-friendly interface

## Tech Stack

* React.js
* Node.js
* Express.js
* MongoDB
Create a .env file and add your MongoDB connection string and other required environment variables.

This project was built to practice full-stack development with the MERN stack and gain hands-on experience with authentication, APIs, database management, and modern web application architecture.
